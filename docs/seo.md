# SEO

Pages, services, industries, resources, and case studies include SEO title, meta description, canonical URL, Open Graph fields, noindex/nofollow controls, sitemap inclusion, sitemap priority, internal keywords, and schema type selection.

Dynamic files:

- `/sitemap.xml` includes published public content.
- `/robots.txt` excludes admin, setup, private form endpoints, and Payload API routes.

Analytics:

- GA4 and GTM IDs may be configured in CMS or public environment variables.
- Private analytics credentials are not supported in the frontend.
- Search Console verification can be set in Site Settings.

Review schema must only be emitted for real published testimonials with permission confirmed.
